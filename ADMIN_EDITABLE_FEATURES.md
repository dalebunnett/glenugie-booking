# Admin Editable Features

All pricing, rates, and booking rules are now fully editable through the Admin Dashboard.

## 📍 Access Admin Dashboard

**URL:** `https://your-site.com/admin`

**Login:** Use your admin credentials to access the dashboard.

## 📋 Booking Rules (Fully Editable)

### General Booking Settings

**Advance Booking Window**
- Minimum days in advance: Default 1 day (editable)
- Maximum days in advance: Default 365 days (editable)

**Stay Duration**
- Minimum nights: Default 1 night (editable)
- Maximum nights: Default 90 nights (editable)

**Same-Day Bookings**
- Allow same-day check-in/check-out: Yes/No toggle
- Cutoff time for same-day bookings: Default 14:00 (editable)

---

### Peak Season Rules

Add multiple peak season periods with custom minimum stay requirements:

**Example Peak Seasons:**
- Summer: July 1 - August 31 (2 night minimum)
- Christmas/New Year: December 20 - January 5 (3 night minimum)

**Editable Fields:**
- Start date
- End date
- Minimum nights required
- Add/remove peak seasons as needed

---

### Blocked Dates

**Specific Blocked Dates**
- Add individual dates when bookings are not allowed
- Example: December 25, 2026 (Christmas Day)

**Blocked Date Ranges**
- Add date ranges with optional reasons
- Example: December 24-26, 2026 (Christmas Holiday)

**Features:**
- Add/remove dates and ranges
- Optional reason field for each range
- Dates automatically blocked in booking calendar

---

### Day of Week Restrictions

**Allowed Check-In Days**
- Select which days of the week customers can check in
- Leave all unchecked to allow all days
- Example: Only allow check-ins on Saturdays and Sundays

**Allowed Check-Out Days**
- Select which days of the week customers can check out
- Leave all unchecked to allow all days
- Example: Only allow check-outs on Saturdays and Sundays

---

### Cancellation Policy

**Full Refund Period**
- Days before check-in for 100% refund
- Default: 14 days
- Editable from 0-90 days

**Partial Refund Period**
- Days before check-in for partial refund
- Default: 7 days
- Editable from 0-90 days

**Partial Refund Percentage**
- Percentage refunded during partial period
- Default: 50%
- Editable from 0-100%

**Example Policy:**
- Cancel 14+ days before: **100% refund**
- Cancel 7-14 days before: **50% refund**
- Cancel less than 7 days before: **No refund**

## 🎯 How to Edit

### Edit Rates
1. Go to Admin Dashboard → **Rates** tab
2. Click **"Edit Rate"** on any accommodation type
3. Update base price and/or additional pet rate
4. Click **"Save Changes"**

### Edit Payment Settings
1. Go to Admin Dashboard → **Rates** tab
2. Scroll to **Payment Settings** card
3. Click **"Edit Settings"**
4. Update deposit amount and/or full payment days
5. Click **"Save Payment Settings"**

### Edit Booking Rules
1. Go to Admin Dashboard → **Booking Rules** tab
2. Click **"Edit"** on any rule section:
   - General Settings
   - Peak Seasons
   - Blocked Dates
   - Day Restrictions
   - Cancellation Policy
3. Make your changes
4. Click **"Save Changes"**
